# AGENTS.md

## 目的
このドキュメントは、AI エージェント共通の作業方針を定義します。

## 基本方針
- 会話と言語: 日本語を使用する。
- コード内コメント: 日本語で記載する。
- エラーメッセージ: 英語で記載する。
- コミット規約: Conventional Commits に従い、`<description>` は日本語で記載する。
- 日本語と英数字の間には半角スペースを挿入する。

## 判断記録のルール
- 重要な設計判断や変更を行う際は、判断理由、検討した代替案、前提条件を明示する。

## 開発手順
1. プロジェクトの構造と依存関係を `package.json` から理解する。
2. `pnpm install` で依存関係をセットアップする。
3. 変更を実装し、docstring を日本語で付与する。
4. `pnpm lint` および `pnpm test` を実行して品質を確保する。

## セキュリティ / 機密情報
- `data/` ディレクトリ配下のファイルや環境変数の機密情報をコミットに含めない。
- ログ出力に認証情報が含まれないよう注意する。

## リポジトリ固有
- pixiv のブックマーク監視を行う TypeScript プロジェクトである。
- `@book000/node-utils` の Logger と Discord クラスを標準的に使用する。
